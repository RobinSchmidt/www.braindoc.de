23.11.2005: Ich führe eine Versions-Nummerierung für meine PlugIns ein. Das ertse Feld ist die Hauptversionsnummer, die sich nur ändert, wenn die neue Version so viele neue Features hat, das es eigentlich um ein ganz anderes PlugIn handelt. Die frühere Version wird in diesem Fall witherhin verfügbar bleiben. Das zweite Feld ändert sich immer dann, wenn wichtige neue Features implementiert worden sind. In diesem Fall wird die frühere Version durch die neue ersetzt. Das dritte und letzte Feld letztendlich enthält die Release-Nummer, die sich immer dann ändert, wenn Bugs gefixt wurden oder sich einige kleine Details (z.B. an der grafischen Benutzeroberfläche) ändern.
